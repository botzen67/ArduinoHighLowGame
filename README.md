- Game variables can be modified in the `setup` function in `sketch/sketch.ino`

# Try it out
You can try out the code project without an Arduino

- 1: Go to https://wokwi.com/projects/new/arduino-uno in a new tab
- 2: Copy/paste `sketch/sketch.ino` into the `sketch.ino` tab
- 3: Copy/paste `diagram.json` into the `diagram.json` tab
- 4
  - Go to the Library Manager tab
  - Press the "+" button
  - Enter `Keypad` in the search box
  - Select the option named `Keypad`
  - In the top-left of the right panel, press the green play button
